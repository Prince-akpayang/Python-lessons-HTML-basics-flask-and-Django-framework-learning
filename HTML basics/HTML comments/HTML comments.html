HTML Comments

Introduction to Comments

HTML comments are used to add notes or comments within the HTML code. They are not displayed on the webpage.

For example:

<!-- This is an HTML comment -->

Usage and Syntax

Comments in HTML are written using the syntax <!-- comment -->. They can span multiple lines.

For example:

<!--
 This is a multi-line
 HTML comment
-->

Placement and Purpose

Comments can be placed anywhere within the HTML code for documentation, temporary disabling of code, or adding notes for developers.

<div class="main">
 <!-- This is the main content area -->
 <p>Content goes here</p>
</div>
