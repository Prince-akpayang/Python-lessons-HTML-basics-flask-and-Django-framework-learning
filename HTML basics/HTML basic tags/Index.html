HTML Basic Tags

Introduction to HTML Tags

HTML tags are used to define elements on a webpage. They are enclosed in angle brackets, like <tagname>.

For example:

<p>This is a paragraph</p>
<img src="image.jpg" alt="Description">

Commonly Used Basic Tags

There are several commonly used HTML tags:

  • <p> - Defines a paragraph
  • <a> - Creates a hyperlink
  • <img> - Embeds an image

Examples and Usage

Let's explore how some basic tags are used in HTML:

<h1>Heading 1</h1>
<p>This is a paragraph</p>
<a href="#section">Jump to Section</a>

Explanation of Tag Attributes

HTML tags can have attributes that provide additional information or functionality to elements:

<a href="https://example.com" target="_blank">Link to Example</a>
<img src="image.jpg" alt="Description">

Nesting Tags

HTML tags can be nested inside each other:

<div>
 <p>This is a nested paragraph inside a div</p>
</div>
