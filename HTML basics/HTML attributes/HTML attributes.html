HTML Attributes

Introduction to Attributes

HTML attributes provide additional information or functionality to elements and are added within the opening tag of an element.

For example:

<a href="https://example.com" target="_blank">Link to Example</a>
<img src="image.jpg" alt="Description">

Commonly Used Attributes

There are various attributes used with HTML elements:

  • href - Specifies the URL in an anchor tag (<a>)
  • src - Specifies the source of an image in an image tag (<img>)
  • alt - Specifies an alternate text for an image

Usage and Examples

Let's explore how attributes are used in HTML:

Importance and Values

Attributes can be crucial for providing essential information or defining behavior for HTML elements:

<input type="text" id="username" name="username" required>
<button onclick="submitForm()" disabled>Submit</button>
